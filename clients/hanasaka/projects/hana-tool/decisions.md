# 決定事項・確定仕様 — HANAツール

## 評価・スキル機能（2026-05-09 決定）

### 方針

- **人事評価機能を HANAツール に組み込む**（システム化）
- 検討時、スプレッドシート運用も候補に挙がったが、**システム化のメリット**（履歴管理・自動集計・運用負荷軽減・全社員横断の一元管理）を優先

### 評価方式

| 項目 | 内容 |
|------|------|
| 形式 | スキルチェックシート（質問に対する評価） |
| 評価段階 | **3段階** |
| 実施頻度 | **半期に1度** |
| 評価主体 | **社員の自己評価 ＋ 社長によるレビュー** |
| 結果の用途 | **教育・意識向上 ＋ 人事査定** の両方 |

### 設計思想

- 質問項目に **「会社として身に着けてほしい項目」** を含めることで、社員の自然なベース向上を狙う
- スキル可視化 ＋ 育成・査定の基盤として機能させる
- 2026-05-08 「スキルシート構想」（→ `notes.md`）と一体の取り組み
