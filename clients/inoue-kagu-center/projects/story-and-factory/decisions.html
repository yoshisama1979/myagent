商品ページ プルダウン初期表示の制御仕様（2024-10-04 シソーラス由来）

制御ファイル

  • 編集箇所：assets/js/items.detail.js の 45行目からのswitch文
  • 分岐キー：tid（アイテムID = URL /item/{tid}/ の数字）

chg_arr の指定ルール

  • 形式：chg_arr = [[セレクトindex, 選択肢index], ...]
  • セレクトが2つある場合：1個目 = index 0、2個目 = index 1
  • 選択肢のindexも 0 始まり
  • 選択肢が 0 のものを指定したい場合は 該当行を省略可（デフォルトで 0 が選ばれるため）
  • すべて省略（chg_arr = []）すると 登録SKU最上位がそのまま表示される

記載例（シソーラス共有）

「1個目を選択肢3、2個目を選択肢2」にしたい場合：

chg_arr = [
  [0, 2],
  [1, 1]
]

「2個目のみ、選択肢5番目（=index 4）を初期表示」にしたい場合：

chg_arr = [
  [1, 4]
]

注意：シソーラス記載例では「選択肢3 → index 2」のように、選択肢の番号（序数・1始まり）と index（0始まり）が1ずれて記載されている。クライアントから「選択肢N」と指定された際は、序数表記か index 表記かを実装前に実機で確認すること。
