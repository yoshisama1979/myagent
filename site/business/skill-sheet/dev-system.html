システム開発スキル

PHP/Laravel、DB、API、サーバー・インフラ、セキュリティなどシステム開発領域のスキル。

2026-05-24 ドラフト（社長レビュー待ち）

対象スコープ

業務システム・API・バックエンドを扱う システム開発担当者向けのシート。営業職・Web制作専任は評価対象外でOK。

評価ルール: PC操作スキルと同じ（5段階評価／重要度別 段階的進行）

重要度の凡例: A 基礎（必須） B 業務効率（推奨） C 専門（応用）

1. PHP / Laravel（7項目）

重要度 項目
APHP の基本構文（変数、関数、配列、クラス）を使える
ALaravel の ルーティング・コントローラ・ビュー（MVC）の基本を理解している
AEloquent ORM の基本（モデル・リレーション）を理解している
BMiddleware、Service、Repository パターンを使える
BQueue、Job、Scheduler を活用できる
CLaravel のテスト機能（PHPUnit、Pest）を実装できる
Cパッケージ作成・依存管理（Composer）を理解・運用できる

2. データベース（SQL / 設計）（7項目）

重要度 項目
ASELECT、INSERT、UPDATE、DELETE の基本 SQL を書ける
AJOIN（INNER、LEFT）を使える
Aテーブル設計（主キー・外部キー・INDEX）の基本を理解している
Bトランザクション・ロックを適切に使える
BN+1問題 を理解し、回避（eager loading）できる
Bマイグレーション・シーダーで DB を管理できる
Cパフォーマンスチューニング（実行計画、INDEX 最適化、SLOW QUERY 分析）ができる

3. API 設計・実装（6項目）

重要度 項目
AREST API の基本（HTTPメソッド、ステータスコード）を理解している
AJSON レスポンスを適切に設計できる
B認証（Token、OAuth、API Key）を実装できる
Bバリデーション・エラーハンドリングを統一的に実装できる
BAPI ドキュメント（OpenAPI / Swagger 等）を書ける
C外部 API 連携（決済、SNS、AI、Chatwork 等）を実装できる

4. サーバー・インフラ（6項目）

重要度 項目
ASSH でサーバーに接続できる
ASCP / SFTP / Git でデプロイができる
Bnginx / Apache の基本設定を変更できる
Bcron でバッチを設定できる
Bログの確認・調査（tail、grep、less）ができる
CVPS・クラウド（さくら、AWS、ConoHa等）の構築・運用ができる

5. セキュリティ（5項目）

重要度 項目
Aパスワードを 平文で保存しない（ハッシュ化を理解している）
ASQLインジェクション・XSS を回避する書き方ができる（プレースホルダ、エスケープ）
BCSRF 対策を理解し実装できる
BHTTPS・SSL証明書の管理ができる（Let's Encrypt 等）
C脆弱性診断ツール・セキュリティアップデートを継続的に実施できる

6. システムテスト・運用（4項目）

重要度 項目
A動作確認時、複数ユーザー・複数ケースで検証する
Bステージング環境で本番デプロイ前に確認する
Bバックアップ・リストア手順を整備している
C監視・アラート設定で異常を検知できる（死活監視、エラー通知）

項目数 集計

重要度 項目数
A 基礎（必須）13
B 業務効率（推奨）15
C 専門（応用）7
合計35
