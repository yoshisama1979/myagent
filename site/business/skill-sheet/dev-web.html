Web制作スキル

HTML/CSS/JavaScript からレスポンシブ・SEO・WordPress まで、Web制作領域のスキル。

2026-05-24 ドラフト（社長レビュー待ち）

対象スコープ

Web制作（HP・LP・キャンペーンサイト等）に関わる技術スキル。Web制作担当者向けのシート。営業職等は評価対象外でOK。

評価ルール: PC操作スキルと同じ（5段階評価／重要度別 段階的進行）

重要度の凡例: A 基礎（必須） B 業務効率（推奨） C 専門（応用）

1. HTML / マークアップ（6項目）

重要度 項目
AHTML の基本構造（doctype, html, head, body）を理解している
AセマンティックHTML を理解し、適切な要素を選べる（header, main, article, section, nav 等）
A適切な見出しレベル（h1〜h6）の階層を組める
Bメタタグ（OGP、Twitter Card 等）を適切に設定できる
B構造化データ（Schema.org / JSON-LD）を意識できる
CARIA 属性を使ったアクセシブルなマークアップができる

2. CSS / スタイリング（8項目）

重要度 項目
ACSS セレクタの基本（class、id、子要素、隣接、疑似要素等）を使える
AFlexbox でレイアウトを組める
A文字サイズ・色・余白の設計ができる（rem/em の使い分け含む）
BGrid Layout でレイアウトを組める
BCSS変数（カスタムプロパティ）を活用できる
Bメディアクエリでレスポンシブ対応できる
CTailwind CSS / Sass / PostCSS 等のモダンツールを使いこなせる
Cアニメーション・トランジションを適切に実装できる

3. JavaScript / インタラクション（6項目）

重要度 項目
ADOM 操作の基本（querySelector、addEventListener 等）ができる
Aフォームのバリデーション・送信処理を実装できる
A非同期処理（fetch、Promise、async/await）の基本を理解している
BjQuery と素のJSの両方を場面で使い分けられる
Bモダンライブラリ（Vue、React、Alpine.js 等）の基本理解がある
Cパフォーマンス（遅延読込、レンダリング最適化、debounce/throttle）を意識できる

4. レスポンシブ・モバイル対応（5項目）

重要度 項目
Aスマホ・タブレット・PC でレイアウトを最適化できる
Aviewport の設定を適切に行える
Bタッチデバイス特有の操作（ホバー無し、タップ領域）を考慮できる
B画像の最適化（srcset、loading=lazy、WebP）ができる
CPWA、オフライン対応、Service Worker の基本を知っている

5. アクセシビリティ（4項目）

重要度 項目
Aalt 属性、label と input の紐付けが適切にできる
Bキーボード操作で操作可能なフォーム・ナビゲーションを作れる
Bコントラスト比・文字サイズを意識できる
Cスクリーンリーダー対応・WCAG ガイドラインを意識した実装ができる

6. SEO・パフォーマンス（5項目）

重要度 項目
Atitle、description、構造化された URL を意識できる
B画像最適化、不要なライブラリ削除でページ速度を上げられる
BPageSpeed Insights、Lighthouse の結果を読み解ける
CCore Web Vitals（LCP、INP、CLS）を理解し、改善できる
Cサーバーサイドのキャッシュ・CDN 活用を提案できる

7. CMS（WordPress）（5項目）

重要度 項目
AWordPress の管理画面で記事・固定ページ・メニューを編集できる
Aテーマ・プラグインの追加・更新ができる
B子テーマ を使った安全なカスタマイズができる
Bfunctions.php、テンプレートファイルを修正できる
Cカスタム投稿タイプ・カスタムフィールド（ACF 等）を実装できる

8. クロスブラウザ・実機確認（3項目）

重要度 項目
AChrome、Edge、Safari、Firefox での表示確認をする習慣がある
A実機（iPhone / Android）で確認する習慣がある
Bブラウザの開発者ツールで他端末をエミュレートできる

項目数 集計

重要度 項目数
A 基礎（必須）15
B 業務効率（推奨）15
C 専門（応用）12
合計42
