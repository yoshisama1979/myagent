2. 想定通り / 想定外（10分）

✅ 想定通りだったこと（≒継続したいこと）

  • 2期連続の黒字維持（経常利益132万、当期純利益104万）
  • 自己資本比率 73.1%（前期 85.8% → -12.7pt） でも依然「非常に健全」レベル（中小企業平均 約40%）
  • キャッシュ残高 866万円 = 月販管費の約4ヶ月分 を確保。守りは固い
  • 既存クライアントとの継続案件は安定（売掛金回収トップ：アースレイズ・サンヒル・ユニバーサルバリュー）

📈 上振れた要因（再現したい）

  • 「外注 → 内製化」シフトの大幅進展：業務委託料が前期比 -55%（-496万、903万→407万）
  • 社員が外注業務を内製対応できる範囲が広がった = 組織能力の向上（給料賃金は +22%／+130万）
  • 業務委託の純減（外注減 - 給料増）= ネット 366万円のコスト構造改善ポテンシャル
  • 内製化が定着すれば、同じ売上でも利益が出やすい体質に。FY2026は「内製化の収穫期」として位置付け可能

📉 下振れた要因（避けたい）

  • 営業利益が前期比 -79.5%（386万 → 79万）と大幅減。営業利益率 12.3% → 3.0%（-9.4pt）
  • 売上が前期比 -15%（3,129万 → 2,667万）
    └ 主因：システム開発案件の 単価ムラ（大型案件の有無で年商が振れる構造的リスクが顕在化）
  • 大型案件依存リスク：売掛金が前期比 +181%（148万 → 416万）。1〜2案件の決済タイミングで年商が振れる
  • 車購入のキャッシュ・利益インパクト：固定資産+388万、新規借入+430万、減価償却+64万／年（FY2026以降も継続）
  • キャッシュは前期比 -232万円。投資・借入返済の流出が大きかった

3. FY2026への接続（10分）

確定済の経営方針との接続

2026-05-08 確定：システム制作・運用をメイン事業へ転換、ストック収益基盤を確立する

→ FY2025 の数字（利益率の低さ、健全な財務）はこの方針の 正しさを裏付ける：

  • スポット案件中心では利益率を上げにくい構造が数字に出ている
  • 自己資本比率が高い＝攻めに転じる体力はある
  • 4ヶ月分のキャッシュ＝慎重に攻める前提なら新規投資余地あり

FY2026 固定費の試算（社員増を加味）

FY2025 販管費合計¥25,878,745
+ 新入社員 総コスト（年俸 ¥4,500,000 + 法定福利・福利・備品等）+約 ¥5,500,000
+ 車両借入の返済（金利想定）+約 ¥500,000
FY2026 想定固定費（概算）約 ¥31,000,000〜32,000,000

※ 内製化進展による業務委託料の追加削減や、既存社員の単価向上で固定費は上下する。あくまで上振れ前提の保守的試算。

→ 目標ツリー（goals.html） の G0 直下に FY2026 売上目標として反映。

最重点論点：ストック収益比率の向上

FY2025 で顕在化した「売上ムラ」問題は、ストック収益化が王道の解

  • システム開発の単価ムラ → 1〜2案件で年商が振れる構造
  • 解：月額運用契約・HANAチャット販売 でフロー収益を支える基盤を作る
  • 目標：FY2026 末でストック収益比率 20%以上（売上 3,300万 のうち 660万以上）
  • 関連：G1（HANAチャット販売拡大） と G2（システム受注件数増加） の月額契約サブ目標

4. 残された論点（継続検討）

確定済

  • ✅ FY2026 売上目標：3,000 / 3,300 / 3,700 万円（3段階）
  • ✅ 最重点論点：ストック収益比率の向上
  • ✅ 経営方針：システム制作・運用へ転換（2026-05-08 既定）

継続検討（FY2026 中に詰める）

  1. 大型案件依存の解消策：案件サイズの分散化、月額契約への切り替え提案、安定収益層の厚み増し
  2. 新入社員の活用設計：新人 → 既存業務引き継ぎ、既存社員 → 新規・高単価へシフトする業務移管計画
  3. 車購入の活用計画：営業訪問数・出張・配送等での具体活用を可視化し、投資回収の根拠を明確化
  4. 利益率5%以上に持っていく一手：単価向上／工数削減／固定費見直しのどれを主軸にするか
  5. 新規4プロジェクト（井上家具・箕面・V-League・よしだ歯科）の収益構造：スポット中心？保守付き？
  6. HANAチャット月額販売の立ち上げ時期：FY2026 のどの時点で売上計上を開始できるか
  7. 利益が出るうちに投資すべき領域：人材／ツール／営業／ブランド

5. 結論・次のアクション（会議後追記）

FY2025 の総括

「売上下落と利益急減という見かけの厳しさの裏で、外注削減＝内製化が大きく進み、組織能力が一段階上がった年」

FY2026 は 内製化の収穫期 と位置付け、ストック収益基盤の構築に踏み出す。財務体質（自己資本比率73.1%、キャッシュ4ヶ月分）は新たな投資・挑戦を支える土台として十分。

FY2026 へ持ち越す宿題

  1. ストック収益の数値定義（何をストックとカウントするか）と目標比率（20%以上）を kpi.html で設計
  2. 大型案件依存の解消策を G2 配下のサブ目標として明文化
  3. 新入社員の業務移管計画と既存社員のシフト先（高単価・新規）を整理
  4. HANAチャット月額販売の立ち上げ計画を G1 配下に具体化（時期・初期ターゲット・価格）
  5. 車購入の活用ROIを半期で振り返れるよう、営業訪問・出張等の指標を設計
