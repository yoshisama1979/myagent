# モード別ルールは `rules/modes/` へ移設済み（2026-07-24）

このディレクトリ（`.claude/rules/`）は全ファイルがセッション開始時に自動ロードされ、毎回約300KBの固定コストになっていたため、モード別ルール15本を **`rules/modes/`（自動ロードされない置き場）** へ移した。

- ルールの一覧と読みどきは **CLAUDE.md の「ルールファイル一覧」** を参照。
- 各モードはスラッシュコマンド（`.claude/commands/*.md`）冒頭の「まず読む」で自分のルールを必要時に読む。
- 古いパス `.claude/rules/<name>.md` を見かけたら `rules/modes/<name>.md` に読み替える。
