# /marketing — マーケティング部ルーター

このコマンドはマーケティング・集客・SNS関連のタスクを処理する。

## 担当エージェント
- **Hina**（`agents/hina-marketing.md`）— 集客・SNS・コンテンツ

## 処理フロー

1. Agent toolでHinaを起動する際、以下を含める:
   - `agents/hina-marketing.md` の内容
   - `guidelines/company-overview.md`
   - `guidelines/brand-guidelines.md`
   - `guidelines/output-standards.md`
   - 具体的なタスク指示

## 対応タスク例
- マーケティング戦略の立案
- SNS投稿案の作成
- コンテンツ企画
- 広告運用の提案
- メールマーケティング施策

## 連携が多いケース
- LP制作が必要 → Sakura（WEB制作部）と連携
- SEOデータが必要 → Yui（WEB運用部）と連携
- 市場調査が必要 → Shun（リサーチ部）と連携
