クライアント情報を取得します。

## キーワード検索（推奨）

ユーザーが特定のクライアントを指定している場合:
```
bash bin/hana-api.sh search "キーワード"
```
- 部分一致で検索。「株式会社」の有無を気にせず検索可能
- カンマ区切りでOR検索可（例: `"トヨペット,はなさか"`）
- クライアント → プロジェクト → 案件（works）まで一括取得

## 全件取得

全クライアントを確認したい場合:
```
bash bin/hana-api.sh clients
```

結果を見やすく整理して表示してください:
- クライアント名、プロジェクト一覧
- 検索時は案件（works）も含めて表示
