クライアント・プロジェクト・案件情報を取得します。

## 取得手順（キャッシュファースト）

**1. まずキャッシュを確認**

`data/clients-cache.json` を読み、キーワードに部分一致するクライアント・プロジェクト・案件を探す。

**2. キャッシュにヒットした場合**

キャッシュの情報を使って回答する。API呼び出しは不要。

**3. キャッシュにない場合のみAPI呼び出し**

```
bash bin/hana-api.sh search "キーワード"
```

- 部分一致で検索。「株式会社」の有無を気にせず検索可能
- カンマ区切りでOR検索可（例: `"トヨペット,はなさか"`）

**4. API取得後、キャッシュを更新**

APIから取得したデータで `data/clients-cache.json` を更新する:
- 既存の `client_id` は最新データで上書き
- 新規の `client_id` は追加
- `updated_at` を現在日時（ISO 8601形式）に更新
- 他のクライアントのデータは削除しない（マージ）

## キャッシュファイル形式

```json
{
  "updated_at": "2026-04-13T12:00:00+09:00",
  "clients": [
    {
      "client_id": 2,
      "client_name": "大阪トヨペット",
      "projects": [
        {
          "project_id": 51,
          "project_name": "おもいでフォト",
          "works": [
            { "work_id": 140, "work_name": "おもいでフォトリニューアル" }
          ]
        }
      ]
    }
  ]
}
```

## 全件取得

全クライアントを確認したい場合（キャッシュ無視）:
```
bash bin/hana-api.sh clients
```

## 結果表示

- クライアント名、プロジェクト一覧
- 案件（works）も含めて見やすく整理
- キャッシュから取得した場合はその旨を明示
