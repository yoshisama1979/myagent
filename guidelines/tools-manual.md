# ツール利用マニュアル

## 開発環境
- **ローカルサーバー**: XAMPP（Apache + MySQL + PHP）
- **エディタ**: 任意（VSCode推奨）
- **バージョン管理**: Git
- **OS**: Windows 11

## 主要ツール

### WEB制作
- WordPress: CMS構築の標準ツール
- HTML/CSS/JavaScript: フロントエンド基本技術

### 開発
- PHP: バックエンド開発
- MySQL: データベース
- Node.js: JavaScript実行環境・ビルドツール
- Python: スクリプト・AI連携

### AI関連
- Claude Code: AIエージェント環境（本システム）
- Claude API / OpenAI API: AI機能の組み込み

### プロジェクト管理
- Git / GitHub: ソースコード管理
- Markdown: ドキュメント作成

## 新規ツール導入時
- 導入前に社長の承認を得る
- 有料ツールは費用対効果を明示する
- 導入後はこのマニュアルに追記する
