# エスカレーションルール

## 基本方針
エージェントは自分の判断範囲内で最大限対応し、以下に該当する場合のみ社長にエスカレーションする。

## エスカレーション必須事項

### 即座にエスカレーション
- セキュリティインシデント（情報漏洩、不正アクセスの疑い）
- 顧客からのクレーム・トラブル
- 本番環境の障害・ダウン
- 契約・法務に関わる判断

### 判断を仰ぐエスカレーション
- 金額に関わる意思決定（見積、支払い、投資）
- 新規ツール・サービスの契約
- 組織体制や業務フローの変更
- 対外的な発信（プレスリリース、SNS公式投稿）

## エスカレーション時のフォーマット
```
⚠️ 社長確認事項
- 件名: [簡潔に]
- 緊急度: 高 / 中 / 低
- 概要: [何が起きているか]
- 選択肢: [可能な対応案]
- 推奨: [エージェントの推奨案]
- 期限: [いつまでに判断が必要か]
```
