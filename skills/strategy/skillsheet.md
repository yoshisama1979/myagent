### SP-01: 事業戦略立案

市場環境・競合動向・自社の強みを総合的に分析し、事業の方向性と勝ち筋を定義する戦略を策定する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 上長の指導のもとで、戦略立案に必要な情報収集ができる | 経営戦略の基本フレームワーク（SWOT、3C、5F）を書籍で学習する |
| Lv.2 | 指定されたフレームワークを使い、定型的な戦略分析資料を作成できる | 自社の過去の戦略資料を読み込み、分析の型を習得する |
| Lv.3 | 市場環境・自社状況を自律的に分析し、戦略仮説を立案・提案できる | 他社の戦略事例を研究し、自社への応用を検討する |
| Lv.4 | 複数の戦略オプションを比較評価し、経営層への提言をリードできる | 戦略コンサルティング手法を学び、社内勉強会を主催する |
| Lv.5 | 業界構造の変化を先読みし、中長期的な事業転換戦略を設計できる | 業界カンファレンスでの登壇・外部アドバイザリーへの参加 |

### SP-02: 中長期計画策定

3〜5年の時間軸で事業ロードマップを設計し、年度ごとのマイルストーンと達成基準を定義する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 上長の指導のもとで、中長期計画の構成要素やフォーマットを理解できる | 過去の中長期計画書を精読し、構造を把握する |
| Lv.2 | 指定されたフォーマットに沿って、計画素案のドラフトを作成できる | 事業計画書の作成手法に関する研修を受講する |
| Lv.3 | 市場予測・リソース制約を考慮し、実現可能な中長期計画を自律的に策定できる | 財務モデリングの基礎を習得し、計画の数値根拠を強化する |
| Lv.4 | 各部門と連携し、全社横断の中長期計画を統合・調整してまとめられる | 部門横断プロジェクトをリードし、計画策定プロセスを改善する |
| Lv.5 | 不確実性を織り込んだシナリオプランニングを設計し、経営判断を支援できる | シナリオプランニング手法の専門研修・外部事例研究に取り組む |

### SP-03: KPI設計・管理

事業目標を測定可能な重要指標（KPI/KGI）に分解し、モニタリング体制を構築・運用する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 上長の指導のもとで、既存KPIの定義とデータ収集方法を理解できる | KPI・KGI・OKRの基本概念を書籍・オンライン教材で学ぶ |
| Lv.2 | 定義されたKPIに基づき、定期的なモニタリングレポートを作成できる | BIツールの基本操作を習得し、ダッシュボード作成を練習する |
| Lv.3 | 事業目標からKPIツリーを設計し、部門ごとの指標に落とし込める | OKRフレームワークの実践事例を研究し、自社への導入を検討する |
| Lv.4 | KPI運用プロセスを改善し、各部門のKPI達成を支援・指導できる | データドリブン経営の手法を学び、社内のKPI文化を醸成する |
| Lv.5 | 事業環境の変化に応じてKPI体系を再設計し、経営の意思決定精度を向上させられる | 先進企業のパフォーマンス管理システムを調査し、ベストプラクティスを導入する |

### SP-04: 経営分析

財務諸表・事業データ・市場データを多角的に分析し、経営判断に資する示唆やレポートを作成する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 上長の指導のもとで、売上・コスト等の基本データを集計・整理できる | 財務諸表の読み方（PL/BS/CF）の基礎を学習する |
| Lv.2 | 定型的な経営レポート（月次報告等）を一人で作成できる | Excelでの財務分析・グラフ作成スキルを強化する |
| Lv.3 | 複数データソースを組み合わせた多角的な分析を行い、課題と示唆を導出できる | 管理会計・原価計算の知識を深め、分析の精度を向上させる |
| Lv.4 | 経営分析の手法・プロセスを標準化し、チームの分析力を底上げできる | データ分析ツール（BI・統計ソフト）の活用を推進し、勉強会を開催する |
| Lv.5 | 経営環境の変化を定量的に予測し、シミュレーションに基づく戦略提言ができる | 統計学・計量経済学の専門知識を習得し、予測モデルを構築する |

### SP-06: 経営リソース配分

ヒト・モノ・カネの経営資源を事業優先度に応じて最適配分し、投資対効果を最大化する提案を行う能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 上長の指導のもとで、各部門のリソース状況を整理・可視化できる | 経営資源管理の基本（ヒト・モノ・カネ）を書籍で学ぶ |
| Lv.2 | 過去の実績データに基づき、予算配分案のドラフトを作成できる | 予算策定プロセスに参加し、配分ロジックの実務を経験する |
| Lv.3 | 事業計画と連動した予算・人員配分案を自律的に策定し、根拠を説明できる | ROI・ROIC等の投資指標を活用した分析手法を習得する |
| Lv.4 | 部門間の利害を調整し、全社最適の視点でリソース配分を主導できる | 経営シミュレーションゲーム等でリソース配分の意思決定を訓練する |
| Lv.5 | 将来の事業環境変化を見据えた動的リソース配分戦略を設計できる | 経営学の上級知識（ポートフォリオ理論・オプション理論）を習得する |

### SP-08: 事業ポートフォリオ管理

複数事業・サービスラインの戦略的位置づけを評価し、成長・撤退・投資の判断を支援する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 上長の指導のもとで、各事業の売上・利益等の基本データを整理できる | PPM（プロダクト・ポートフォリオ・マネジメント）の基本を学ぶ |
| Lv.2 | 既存フレームワーク（BCGマトリクス等）に沿って事業分類を行える | 自社の事業構成を整理し、各事業の特性を把握する |
| Lv.3 | 市場成長性・競争優位性を評価し、ポートフォリオの課題と方向性を提案できる | アンゾフマトリクス・GEマトリクス等の応用フレームワークを習得する |
| Lv.4 | ポートフォリオ全体の最適化方針を策定し、各事業責任者と調整をリードできる | M&A・アライアンスの基礎知識を学び、ポートフォリオ組替えの選択肢を拡大する |
| Lv.5 | 業界再編・技術変革を見据えた長期ポートフォリオ戦略を設計し、経営判断を主導できる | グローバル企業のポートフォリオ戦略事例を研究し、専門ネットワークを構築する |
