### AM-01: 経費精算・管理

日常の経費申請を処理し、勘定科目の判断や経費規程に基づく適正チェックを行う。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 経費申請の基本フロー（申請→承認→精算）を説明できる。領収書の必要項目を理解している | 社内経費規程を通読する。勘定科目一覧表を確認し、主要な科目を覚える |
| Lv.2 | 定型的な経費申請（交通費、消耗品費等）の勘定科目を正しく判断し、一人で精算処理ができる | 日商簿記3級の学習を開始する。過去の経費精算データを確認し、科目判断のパターンを把握する |
| Lv.3 | 非定型的な経費（接待交際費、外注費等）の科目判断ができる。経費規程に照らした不備指摘・差し戻し対応を自律的に行える | 日商簿記3級に合格する。税務上の損金算入要件を学ぶ |
| Lv.4 | 経費精算ルールの改善提案ができる。全社の経費傾向を分析し、コスト削減の提言ができる。後輩に科目判断の指導ができる | 日商簿記2級の学習を進める。他社の経費管理ベストプラクティスを調査する |
| Lv.5 | 全社の経費管理方針を策定できる。税制改正に対応した経費規程の改訂をリードできる。外部税理士との協議を主導できる | 税理士科目（簿記論・財務諸表論）の学習に取り組む。経理系カンファレンスに参加する |

### AM-02: 請求書・見積書管理

請求書・見積書の作成、内容チェック、発行・受領管理を正確に行う。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 請求書・見積書の基本項目（宛名、金額、消費税、支払条件等）を理解している。テンプレートに沿って下書きができる | 社内の請求書・見積書テンプレートを確認する。インボイス制度の基本を学ぶ |
| Lv.2 | 案件情報をもとに請求書・見積書を一人で作成できる。受領した請求書の基本的なチェック（金額、振込先、適格請求書要件）ができる | MFクラウド請求書の操作を習熟する。適格請求書等保存方式の要件を正確に把握する |
| Lv.3 | 複雑な案件（分割請求、追加費用、値引き等）の請求書を自律的に作成できる。未入金の管理・督促対応ができる | 債権管理の基礎を学ぶ。電子帳簿保存法の要件を理解する |
| Lv.4 | 請求・入金管理のフローを改善し、未回収リスクを低減できる。後輩にインボイス制度の実務対応を指導できる | 与信管理の手法を学ぶ。請求管理業務の自動化・効率化を検討する |
| Lv.5 | 全社の売掛金・買掛金管理の仕組みを設計できる。取引条件の交渉や与信判断を戦略的にリードできる | 経理実務のセミナーや資格取得に取り組む。業界の商慣行に精通する |

### AM-03: 契約書レビュー

業務委託契約書、NDA、利用規約等の契約書類を読解し、リスク観点でレビューする。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 契約書の基本構成（当事者、目的、期間、報酬、解除条件等）を理解している。NDAの基本的な条項を説明できる | ビジネス法務の入門書を読む。社内で使用する契約書テンプレートを通読する |
| Lv.2 | 定型的な業務委託契約書・NDAの条項を一通り確認し、明らかな不備（日付誤り、金額誤り等）を指摘できる | ビジネス実務法務検定3級の学習を開始する。過去の契約書レビュー事例を確認する |
| Lv.3 | 損害賠償条項、知的財産権の帰属、競業避止義務等のリスク条項を識別し、修正案を提示できる。下請法の適用可否を判断できる | ビジネス実務法務検定2級の学習を進める。下請法・著作権法の基礎を学ぶ |
| Lv.4 | 契約交渉において自社に有利な条件を提案できる。契約書テンプレートの策定・改善をリードできる。後輩のレビューを監督できる | 法務セミナーに定期的に参加する。判例・法改正情報をキャッチアップする |
| Lv.5 | 複雑な契約スキーム（SES、レベニューシェア、SaaS利用規約等）のリスク分析と条件設計ができる。外部弁護士との協議を主導できる | ビジネス実務法務検定1級を目指す。IT関連の法務トレンドを継続的にフォローする |

### AM-04: 月次・年次決算補助

月次の数値整理、年次決算に必要なデータ準備・集計を正確に行う。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 貸借対照表・損益計算書の基本構造を理解している。月次決算の流れ（仕訳入力→残高確認→報告）を説明できる | 日商簿記3級のテキストで財務諸表の基礎を学ぶ。月次決算チェックリストを確認する |
| Lv.2 | 月次の仕訳入力、銀行残高の照合、売上・経費の集計を一人で完了できる。前月比の増減を報告できる | 日商簿記3級に合格する。Excelでの集計・ピボットテーブル操作を習熟する |
| Lv.3 | 減価償却費の計算、前払費用・未払費用の計上、売掛金・買掛金の残高管理を自律的に行える。月次レポートを作成し、異常値を報告できる | 日商簿記2級の学習を進める。管理会計の基礎（予実管理）を学ぶ |
| Lv.4 | 年次決算の一連のスケジュール管理と実務をリードできる。税理士と連携して決算書類を取りまとめられる。後輩に月次決算業務を指導できる | 日商簿記2級に合格する。法人税申告の流れを理解する |
| Lv.5 | 財務分析に基づく経営への提言ができる。会計方針の選択・変更の判断ができる。監査対応を統括できる | 日商簿記1級または税理士科目の学習に取り組む。CFO的視点での財務戦略を学ぶ |

### AM-05: 労務基礎（入退社・社保手続き）

入退社時の社会保険・雇用保険手続き、届出書類の作成を正確に行う。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 社会保険（健康保険・厚生年金）と雇用保険の違いを説明できる。入社時に必要な届出書類の種類を把握している | 社会保険・雇用保険の入門書を読む。社内の入退社手続きマニュアルを確認する |
| Lv.2 | 入社時の資格取得届、退社時の資格喪失届を一人で作成・提出できる。離職証明書の基本的な記入ができる | 実際の届出書類を使って記入練習を行う。e-Govの電子申請操作に慣れる |
| Lv.3 | 算定基礎届、月額変更届の作成ができる。産休・育休に伴う社保手続きを自律的に対応できる。労働保険の年度更新手続きができる | 社会保険労務士試験のテキストで労働保険・社会保険の知識を深める |
| Lv.4 | 複雑な労務ケース（二以上事業所勤務、海外赴任等）の手続きを判断できる。社内の労務手続きフローを改善できる。後輩に手続き業務を指導できる | 社会保険労務士試験の受験を検討する。労務管理クラウドツールの導入・運用を推進する |
| Lv.5 | 全社の労務管理体制を設計・最適化できる。法改正に対応した就業規則の改訂をリードできる。外部社労士との連携を統括できる | 社会保険労務士資格の取得を目指す。労務系セミナーで講師を務める |

### AM-06: 給与計算・勤怠管理

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 給与の基本構成（基本給、手当、控除項目）を理解している。勤怠管理システムへの打刻・申請方法を説明できる | 給与計算の入門書を読む。社内の給与規程・就業規則を確認する |
| Lv.2 | 勤怠データの集計（残業時間、有給取得日数等）を一人で行える。基本的な給与計算（総支給額、社会保険料控除、源泉所得税）ができる | 給与計算実務能力検定の学習を開始する。Excelでの給与計算シミュレーションを行う |
| Lv.3 | 賞与計算、年末調整の実務を自律的に遂行できる。36協定の管理、有給休暇の付与日数管理ができる。住民税の特別徴収手続きができる | 給与計算実務能力検定2級に合格する。労働基準法の労働時間規定を学ぶ |
| Lv.4 | 給与体系の見直し提案ができる。勤怠管理・給与計算の業務効率化をリードできる。税制改正に伴う計算ロジックの更新を行える | 給与計算実務能力検定1級を目指す。人事制度と給与制度の連動を学ぶ |
| Lv.5 | 全社の報酬制度設計に関与できる。複雑な給与体系（裁量労働制、変形労働時間制等）の設計・運用ができる。外部専門家と連携し最適な報酬戦略を策定できる | 報酬制度の専門セミナーに参加する。労働法と税法の両面から最適解を導ける知見を蓄積する |

### AM-08: 法令・税務の基礎知識

会社法、税法、労働関連法規の基礎を理解し、日常業務で適切に適用する。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 法人税・消費税・源泉所得税の基本的な仕組みを説明できる。会社法における取締役会・株主総会の基本的な役割を理解している | 税務の入門書を読む。国税庁のウェブサイトで基本的な税制を確認する |
| Lv.2 | 源泉徴収の対象と税率を正しく判断できる。消費税の課税・非課税・免税の区分ができる。労働基準法の基本的な規定を理解している | ビジネス実務法務検定3級を取得する。年末調整・法定調書の実務を経験する |
| Lv.3 | 法人税の基本的な計算構造（益金・損金）を理解している。税務上の経費処理判断（交際費の損金不算入等）ができる。労働関連法規の改正を把握し対応できる | 税務系の実務セミナーに参加する。法改正ニュースを定期的にチェックする |
| Lv.4 | 税制改正が自社に与える影響を分析し、対応策を提案できる。税理士・社労士との専門的な議論ができる。後輩に法令知識を指導できる | 税理士試験の科目学習を検討する。判例研究を通じて実践的な法令解釈力を養う |
| Lv.5 | 税務戦略（節税対策、組織再編税制等）の立案に関与できる。法令遵守体制の構築を全社レベルでリードできる。外部専門家と対等に議論し意思決定できる | 税理士資格の取得を目指す。税法・会社法の専門書を継続的に学習する |
