### PM-01: 要件整理・スコープ定義

クライアントの要望や課題をヒアリングし、プロジェクトのゴール・スコープ・前提条件・制約を明確に定義する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 上長の同席のもと、ヒアリングシートに沿ってクライアント要望を記録できる | ヒアリングシートの項目と目的を学習する。過去案件の要件定義書を3件以上読む |
| Lv.2 | 定型案件（小規模サイト制作等）のスコープ定義書を一人で作成できる | 要件定義の基本フレームワーク（5W1H、MoSCoW法）を習得する |
| Lv.3 | 複雑な要件を構造化し、スコープ・前提条件・除外事項を漏れなく定義できる | PMBOK等の要件管理手法を体系的に学習する。中規模案件のリードを経験する |
| Lv.4 | 曖昧な要望から本質的な課題を見抜き、最適なスコープを提案できる。後輩の要件定義をレビュー・指導できる | 要件定義ワークショップのファシリテーション技法を習得する |
| Lv.5 | 事業戦略レベルの要件を技術要件に落とし込み、複数フェーズの段階的スコープを設計できる | 大規模案件やDX案件の上流工程をリードする。業界カンファレンスでの情報収集 |

### PM-02: スケジュール策定・工程管理

WBS（作業分解構成図）を作成し、マイルストーン・依存関係を考慮した実現可能なスケジュールを策定・管理する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | テンプレートに沿ってWBSの項目を埋め、ガントチャートを作成できる | WBSの基本構造とガントチャートツール（Backlog等）の操作を習得する |
| Lv.2 | 定型案件のスケジュールを作成し、各タスクに適切な工数と担当者を割り当てできる | 過去案件の実績工数を分析し、見積精度を向上させる |
| Lv.3 | タスク間の依存関係を考慮し、クリティカルパスを意識したスケジュールを自律的に策定できる | クリティカルパス法（CPM）、PERT手法を学習する |
| Lv.4 | バッファ設計を含む堅牢なスケジュールを策定し、遅延発生時の代替計画を即座に提示できる。チームのスケジュール策定力を底上げできる | CCPM（クリティカルチェーン）等の高度なスケジューリング手法を習得する |
| Lv.5 | 複数部門・外部パートナーを含む大規模プロジェクトの全体工程を設計し、経営判断に資するスケジュール戦略を立案できる | 大規模プログラムマネジメントの事例研究、PMP等の資格取得 |

### PM-03: 進捗管理・可視化

プロジェクトの進捗状況をリアルタイムに把握し、課題やボトルネックを早期に検知して関係者に共有する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 指示されたタイミングで各担当者に進捗を確認し、所定のフォーマットに記録できる | 進捗管理ツール（Backlog、Notion等）の基本操作を習得する |
| Lv.2 | 定期的な進捗確認サイクルを自分で回し、遅延を検知して上長に報告できる | EVM（アーンドバリューマネジメント）の基礎概念を学習する |
| Lv.3 | 進捗データをダッシュボード等で可視化し、遅延の原因分析と対策提案を自律的にできる | BIツールやプロジェクト管理ツールの高度な活用法を習得する |
| Lv.4 | プロジェクト横断で進捗を俯瞰し、リソース再配分や優先度調整を主導できる。チームの進捗管理プロセスを改善できる | 組織レベルのプロジェクト管理フレームワーク（PMO運営）を学習する |
| Lv.5 | 経営ダッシュボードレベルのKPI設計を行い、全社のプロジェクトポートフォリオの健全性を可視化・報告できる | ポートフォリオマネジメント、経営指標との連動手法を研究する |

### PM-04: リスク管理

プロジェクトに潜在するリスクを早期に特定・評価し、予防策（回避・軽減）と発生時の対応策（受容・転嫁）を計画・実行する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | リスク管理台帳のテンプレートに沿って、指摘されたリスクを記録できる | リスクの基本分類（技術・スケジュール・コスト・外部要因）を学習する |
| Lv.2 | 過去事例を参考に、定型案件の主要リスクを洗い出し、影響度・発生確率を評価できる | リスクマトリクスの作成方法を習得する。過去の振り返り資料を分析する |
| Lv.3 | 案件の特性に応じたリスクを網羅的に特定し、予防策と対応策を含むリスク管理計画を策定できる | リスク分析手法（FMEA、デシジョンツリー等）を学習する |
| Lv.4 | リスクの兆候を早期に察知し、先手を打った対応を実行できる。チーム全体のリスク感度を向上させる指導ができる | リスクマネジメントの組織的導入手法を習得する。ケーススタディ研修を企画する |
| Lv.5 | 事業レベルのリスク評価を行い、経営判断に資するリスク対応戦略を提言できる。業界のリスクトレンドを先読みできる | エンタープライズリスクマネジメント（ERM）の体系を習得する |

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|

### PM-08: 品質・納品管理

成果物の品質基準を設定し、レビュープロセスの運営から納品・検収までを管理する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | チェックリストに沿って、成果物の基本的な確認（リンク切れ、誤字脱字等）ができる | 納品チェックリストの項目と確認方法を習得する |
| Lv.2 | 定型案件の品質チェックを一通り実施し、修正依頼を適切に各部門へ伝達できる | 品質管理の基本概念（品質計画・品質保証・品質管理）を学習する |
| Lv.3 | 案件特性に応じた品質基準とレビュープロセスを設計し、納品までの品質を自律的に担保できる | レビュー手法（ウォークスルー、インスペクション）を体系的に習得する |
| Lv.4 | 品質管理部（Kaito）と連携し、品質基準の継続的改善を推進できる。納品トラブルの根本原因分析と再発防止策を主導できる | 品質マネジメントシステム（ISO 9001等）の基礎を学習する |
| Lv.5 | 全社的な納品品質の向上戦略を立案し、顧客満足度とプロセス効率の両立を実現できる | TQM（総合的品質管理）、カスタマーエクスペリエンス管理を研究する |

### PM-09: 複数案件の並行管理

同時進行する複数プロジェクトのリソース配分・優先順位を最適化し、全体の効率とQCDを維持する能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 自分が関わる複数タスクの期日と優先度をリスト化し、抜け漏れなく対応できる | タスク管理ツールの活用法、GTD等の個人タスク管理手法を学習する |
| Lv.2 | 2〜3件の小規模案件を並行して担当し、各案件のスケジュールと品質を維持できる | 優先順位付けフレームワーク（アイゼンハワーマトリクス等）を習得する |
| Lv.3 | 5件程度の案件を並行管理し、リソース競合時の優先判断と調整を自律的にできる | リソース管理手法、制約理論（TOC）の基礎を学習する |
| Lv.4 | チーム全体の案件ポートフォリオを管理し、リソースの最適配分と案件間の相乗効果を設計できる | ポートフォリオマネジメント、キャパシティプランニングを習得する |
| Lv.5 | 全社レベルのプロジェクトポートフォリオを最適化し、経営戦略と連動した案件優先度の意思決定を支援できる | PPM（プロジェクト・ポートフォリオ・マネジメント）の高度手法を研究する |

### PM-10: プロジェクト振り返り・改善

プロジェクト完了後の振り返り（レトロスペクティブ）を実施し、教訓を体系的に蓄積してプロセス改善につなげる能力。

| レベル | 行動指標 | 推奨学習アクション |
|-------|---------|------------------|
| Lv.1 | 振り返り会議に参加し、所定のフォーマットに沿って気づきや反省点を記録できる | KPT（Keep/Problem/Try）等の振り返りフレームワークの基本を学習する |
| Lv.2 | 定型案件の振り返りをファシリテーションし、改善アクションを具体的にリスト化できる | 振り返り手法のバリエーション（YWT、4Ls等）を習得する |
| Lv.3 | 振り返りから得た教訓をナレッジベースに体系的に蓄積し、次案件の計画に反映できる | ナレッジマネジメントの基礎、レッスンズラーンドの管理手法を学習する |
| Lv.4 | 部門横断の振り返りをリードし、プロジェクト管理プロセス全体の改善施策を企画・実行できる | 組織学習・プロセス改善（PDCA、DMAIC）の高度手法を習得する |
| Lv.5 | 全社のプロジェクト成功率・効率性を向上させる仕組みを構築し、継続的改善文化を醸成できる | アジャイルコーチング、組織変革マネジメントの先進事例を研究する |
