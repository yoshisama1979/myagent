# YCOM（個人事業）データ保管

社長が運営する **個人事業 YCOM** の経営・財務データを置く場所。
株式会社はなさかとは **別組織** として管理する。

## 背景（経緯）

- 元々 YCOM として個人事業を立ち上げ
- その後「法人成り」ではなく、**別法人として** 株式会社はなさかを新規設立
- 現在は YCOM（個人事業）と 株式会社はなさか（法人）を **並行運営**

## 株式会社はなさかとの違い

| 項目 | 株式会社はなさか | YCOM（個人事業） |
|------|--------------|--------------|
| 組織形態 | 法人 | 個人事業 |
| 会計期間 | **事業年度 4月〜3月**（FY） | **暦年 1月〜12月**（個人事業の決算） |
| 決算様式 | 法人決算（PL/BS/法人税申告書） | 確定申告書（青色申告 想定） |
| データ置き場 | `data/financial/` | `data/ycom/financial/`（本ディレクトリ配下） |

## ディレクトリ構成

```
data/ycom/
├── README.md              # 本ファイル
└── financial/             # 財務データ
    ├── 2024/              # 2024年（1-12月）
    └── 2025/              # 2025年（1-12月）
```

## セキュリティ

- 財務データ本体は `.gitignore` で除外（コミットしない）
- README のみ追跡対象

## 運用

1. 会計ソフトまたはスプレッドシートから関連データをエクスポート
2. `data/ycom/financial/YYYY/` に配置
3. AI（私）が分析、結果を経営判断やレビューに反映

## 今後の整備（要確認）

以下は社長と相談の上で決めていく：

- 使用している会計ソフト（マネーフォワード／freee／弥生／手動 など）
- 事業内容（はなさかとの棲み分け）
- 決算サポートの範囲（確定申告書作成支援／帳簿チェック／経営分析 等）
- YCOM 単独の業績指標（売上・経費・利益）を追跡するか
- YCOM と はなさか の連結的に見たい場合の整理方針
